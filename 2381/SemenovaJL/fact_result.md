## Фактические результаты
На данном этапе выполнения работы имеются следующие результаты: 
1. Сформирована тема научной работы: «Исследование и реализация адаптивных моделей обучения тренеров дельфинов». 
2. Определено понятие адаптивной обучающей системы, рассмотрены преимущества адаптивного обучения в сравнении с традиционным методом, проведено исследование основных принципов построения обучающей системы на основе адаптивной модели обучения.
3. Построена адаптивная модель обучения тренеров дельфинов.
4. Проведено проектирование обучающей программы: составлена схема связей между элементами (блоками) программы и с внешней базой данных, в которой будут храниться текущие и итоговые результаты обучения тренеров дельфинов и обучающий материал. 
5. Выбраны средства разработки: язык написания C#, среда разработки Microsoft Visual Studio; программа для создания анимации Beneton Movie GIF; СУБД Microsoft Access; язык гипертекстовой разметки HTML; инсталлятор Inno Setup.
6. Составлен план реализации обучающей программы и определены основные модули: 
•	модуль регистрации и авторизации пользователей, необходимый для идентификации их в системе;
•	обучающий модуль, хранящий в себе набор заданий по тренировке дельфинов;
•	модуль тестирования, необходимый для проверки знаний, полученных пользователями программы в процессе обучения;
•	модуль редактирования исходных данных программы, касающихся обучающего модуля и модуля тестирования;
•	модуль управления адаптивным обучением;
•	модуль просмотра протоколов обучаемых, хранящий промежуточные и итоговые результаты обучения.
## Планируемые результаты 
В ближайшее время планируется выполнить следующие задачи, необходимые для достижения поставленной цели:
1. Реализация вышеперечисленных модулей программы.
2. Тестирование разработанной программы и анализ полученных результатов.
